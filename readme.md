# Retrograming Wiki API

Projet squelette

## Installation des dépendances Node.js

Se placer à la racine du dossier api : 

`cd ./api`

et exécuter la commande :

`npm i`

## Démarrage de l'architecture Docker

Se placer à la racine du projet et exécuter la commmande : 

`docker-compose up`

Pour arrêter les services Docker : 

CTRL + c
